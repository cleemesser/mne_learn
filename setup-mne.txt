install mayavi extension for jupyter notebook
---------------------------------------------
$ jupyter nbextension install --py mayavi --user
$ jupyter nbextension enable mayavi --user --py

Then in he notebook you can do:
  from mayavi import mlab
  mlab.init_notebook()
